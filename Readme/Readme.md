## 📝 Descripción

Este repositorio contiene la resolución detallada del examen avanzado de **Python y Análisis de Datos**. El proyecto se centra en el estudio de dos conjuntos de datos clave: `Customer Flight Activity.csv` y `Customer Loyalty History.csv`.

El objetivo principal es realizar un ciclo completo de análisis de datos (**EDA**), desde la limpieza y unión de fuentes hasta la visualización estratégica, con el fin de proporcionar insights sobre la fidelidad de los clientes para la aerolínea "Adalab".

-----

## 📊 Flujo del Proyecto

El análisis se divide en tres fases críticas para garantizar la integridad de los resultados:

1.  **Fase 1: Exploración y Limpieza (EDA):** Análisis exhaustivo columna por columna. No solo se gestionan nulos y duplicados, sino que se analiza la distribución estadística (media, mediana, desviación estándar) para comprender la naturaleza de los datos.
2.  **Fase 2: Análisis Estadístico:** Traducción de los datos a lenguaje natural, identificando patrones de comportamiento y tendencias de los clientes.
3.  **Fase 3: Visualización Dinámica:** Creación de gráficos modernos y funcionales que facilitan la interpretación de casos de negocio complejos.
